Проекты Arduino для начинающих

Практические проекты для изучения Arduino с пошаговыми инструкциями

Введение

Лучший способ изучить Arduino — это создавать проекты. В этом модуле мы представляем несколько проектов для начинающих, которые помогут вам освоить основы работы с Arduino, датчиками и исполнительными механизмами. Каждый проект включает список необходимых компонентов, пошаговые инструкции и код.

Проект 1: Светодиодный светофор

Светодиодный светофор

Уровень: Начинающий

Создайте простой светофор с тремя светодиодами (красный, желтый, зеленый), которые загораются в определенной последовательности, имитируя работу настоящего светофора.

Необходимые компоненты:

  • Arduino Uno
  • Макетная плата
  • Красный светодиод
  • Желтый светодиод
  • Зеленый светодиод
  • 3 резистора 220 Ом
  • Соединительные провода

Пошаговая инструкция:

Шаг 1: Подключение светодиодов

Подключите светодиоды к Arduino следующим образом:

  • Красный светодиод: анод (+) через резистор 220 Ом к пину 10, катод (-) к GND
  • Желтый светодиод: анод (+) через резистор 220 Ом к пину 9, катод (-) к GND
  • Зеленый светодиод: анод (+) через резистор 220 Ом к пину 8, катод (-) к GND

Шаг 2: Написание кода

Загрузите следующий код в Arduino:

// Определение пинов для светодиодов
const int redPin = 10;
const int yellowPin = 9;
const int greenPin = 8;

void setup() {
  // Настройка пинов как выходов
  pinMode(redPin, OUTPUT);
  pinMode(yellowPin, OUTPUT);
  pinMode(greenPin, OUTPUT);
}

void loop() {
  // Красный свет (стоп)
  digitalWrite(redPin, HIGH);
  digitalWrite(yellowPin, LOW);
  digitalWrite(greenPin, LOW);
  delay(3000); // Ждем 3 секунды
  
  // Желтый свет (приготовиться)
  digitalWrite(redPin, LOW);
  digitalWrite(yellowPin, HIGH);
  digitalWrite(greenPin, LOW);
  delay(1000); // Ждем 1 секунду
  
  // Зеленый свет (движение)
  digitalWrite(redPin, LOW);
  digitalWrite(yellowPin, LOW);
  digitalWrite(greenPin, HIGH);
  delay(3000); // Ждем 3 секунды
  
  // Желтый свет (приготовиться к остановке)
  digitalWrite(redPin, LOW);
  digitalWrite(yellowPin, HIGH);
  digitalWrite(greenPin, LOW);
  delay(1000); // Ждем 1 секунду
  
  // Цикл повторяется
}

Шаг 3: Тестирование

Загрузите код в Arduino и наблюдайте, как светодиоды загораются в последовательности светофора: красный → желтый → зеленый → желтый → красный и т.д.

Шаг 4: Эксперименты

Попробуйте изменить время задержки для разных сигналов светофора. Можно также добавить кнопку для пешеходного перехода, которая будет прерывать обычный цикл и включать красный свет.

Проект 2: Датчик температуры и влажности

Монитор температуры и влажности

Уровень: Начинающий

Создайте устройство для измерения температуры и влажности с выводом данных на серийный монитор.

Необходимые компоненты:

  • Arduino Uno
  • Датчик DHT11 (или DHT22)
  • Резистор 10 кОм
  • Макетная плата
  • Соединительные провода

Пошаговая инструкция:

Шаг 1: Установка библиотеки

Установите библиотеку DHT через менеджер библиотек Arduino IDE (Скетч → Подключить библиотеку → Управлять библиотеками... → найдите "DHT sensor library" от Adafruit).

Шаг 2: Подключение датчика

Подключите датчик DHT11 к Arduino следующим образом:

  • VCC датчика к 5V Arduino
  • GND датчика к GND Arduino
  • DATA датчика к пину 2 Arduino
  • Подключите резистор 10 кОм между VCC и DATA (подтягивающий резистор)

Шаг 3: Написание кода

Загрузите следующий код в Arduino:

#include "DHT.h"

#define DHTPIN 2     // Пин, к которому подключен датчик
#define DHTTYPE DHT11   // Тип датчика (DHT11 или DHT22)

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  Serial.println("Тест датчика DHT!");
  
  dht.begin();
}

void loop() {
  // Ждем 2 секунды между измерениями
  delay(2000);
  
  // Считываем влажность
  float h = dht.readHumidity();
  // Считываем температуру в градусах Цельсия
  float t = dht.readTemperature();
  
  // Проверяем, корректно ли считаны данные
  if (isnan(h) || isnan(t)) {
    Serial.println("Ошибка чтения с датчика DHT!");
    return;
  }
  
  // Выводим данные в серийный порт
  Serial.print("Влажность: ");
  Serial.print(h);
  Serial.print(" %\t");
  Serial.print("Температура: ");
  Serial.print(t);
  Serial.println(" °C");
}

Шаг 4: Тестирование

Загрузите код в Arduino и откройте серийный монитор (Инструменты → Серийный монитор). Установите скорость 9600 бод. Вы должны увидеть данные о температуре и влажности, обновляемые каждые 2 секунды.

Шаг 5: Расширение проекта

Для расширения проекта можно добавить LCD-дисплей для отображения данных без компьютера или подключить светодиоды, которые будут загораться при определенных условиях (например, красный при высокой температуре).

Проект 3: Ультразвуковой дальномер

Ультразвуковой дальномер

Уровень: Средний

Создайте устройство для измерения расстояния с помощью ультразвукового датчика HC-SR04.

Необходимые компоненты:

  • Arduino Uno
  • Ультразвуковой датчик HC-SR04
  • Макетная плата
  • Соединительные провода
  • Зуммер (опционально)

Пошаговая инструкция:

Шаг 1: Подключение датчика

Подключите ультразвуковой датчик HC-SR04 к Arduino следующим образом:

  • VCC датчика к 5V Arduino
  • GND датчика к GND Arduino
  • Trig датчика к пину 9 Arduino
  • Echo датчика к пину 10 Arduino

Шаг 2: Написание кода

Загрузите следующий код в Arduino:

const int trigPin = 9;
const int echoPin = 10;

// Определение переменных
long duration;
int distance;

void setup() {
  // Настройка Serial порта
  Serial.begin(9600);
  // Настройка пинов
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
}

void loop() {
  // Очищаем trigPin
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  
  // Устанавливаем trigPin в HIGH на 10 микросекунд
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  // Считываем echoPin, возвращает время прохождения звуковой волны в микросекундах
  duration = pulseIn(echoPin, HIGH);
  
  // Рассчитываем расстояние
  distance = duration * 0.034 / 2; // Скорость звука: 0.034 см/мкс, делим на 2 (туда и обратно)
  
  // Выводим расстояние в серийный порт
  Serial.print("Расстояние: ");
  Serial.print(distance);
  Serial.println(" см");
  
  delay(500); // Пауза между измерениями
}

Шаг 3: Тестирование

Загрузите код в Arduino и откройте серийный монитор. Вы должны увидеть измеренное расстояние в сантиметрах. Попробуйте поместить различные предметы перед датчиком на разных расстояниях и наблюдайте за изменениями показаний.

Шаг 4: Добавление звукового сигнала (опционально)

Если вы хотите добавить звуковой сигнал, который будет меняться в зависимости от расстояния, подключите зуммер к пину 3 и GND, затем добавьте следующий код в функцию loop():

// Добавьте в начало скетча
const int buzzerPin = 3;

// Добавьте в setup()
pinMode(buzzerPin, OUTPUT);

// Добавьте в конец loop() перед delay(500)
if (distance < 20) {
  // Если расстояние меньше 20 см, издаем звуковой сигнал
  // Частота и длительность сигнала зависят от расстояния
  int frequency = map(distance, 0, 20, 2000, 500);
  tone(buzzerPin, frequency, 100);
} else {
  // Если расстояние больше 20 см, выключаем звук
  noTone(buzzerPin);
}

Заключение

Не бойтесь экспериментировать и модифицировать проекты под свои нужды. Именно так вы сможете лучше понять, как работает Arduino и электроника в целом.
